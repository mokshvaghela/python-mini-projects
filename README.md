## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/mokshvaghela/python-mini-projects.git
```

2. Navigate to the project folder

```bash
cd python-mini-projects
```

3. Run any project

```bash
python filename.py
```

Replace `filename.py` with the project you want to run.
