# 🪨 Stone Paper Scissors Game

A simple command-line implementation of the classic **Stone, Paper, Scissors** game using Python. The player competes against the computer, which makes random choices each round.

## 🎮 Features

* Play against the computer
* Random computer moves
* User-friendly input
* Winner determination based on game rules
* Option to play multiple rounds *(if implemented)*

## 🛠️ Technologies Used

* Python 3
* `random` module
